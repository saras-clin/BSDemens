  STATUS: Implemented in R/01_build_cohorts.R, build_bs_cohort().
  Filter changed from doddato >= surgery_date to doddato > surgery_date + 30,
  which excludes deaths from day 0 (surgery day) through day 30 inclusive.

CRITICAL-3: NMI dementia predictors must be excluded for Study 1
  STATUS: IMPLEMENTED (exclude_dementia_predictors = TRUE in extract_nmi).
  LOCATION: R/02_extract_outcomes_covariates.R, extract_nmi()
  CONTEXT: The NMI (Kristensen et al., Clin Epidemiol 2022) includes:
    dx_F00_G30 (F00-F03 / G30 dementia diagnoses, weight = 9) and
    rx_N06D (antidementia medications, weight = 11) as predictors.
    In Study 1 where dementia is the OUTCOME, adjusting for these predictors
    in Cox models would introduce circular confounding (near-perfect collinearity
    with the outcome for incident dementia cases). The exclude_dementia_predictors
    parameter was added and main_extraction() now calls extract_nmi with TRUE.
  NOTE: This means the NMI score used in Study 1 is a modified NMI (48 predictors
    instead of 50). This should be noted in the methods section.

CRITICAL-4: Emigration censoring — RESOLVED
  CONFIRMED: VNDS register (load_database("vnds")).
    indud_kode == "U" = emigration (udrejse); haend_dato = emigration date (character "YYYY-MM-DD").
  IMPLEMENTED: get_emigration_dates() in R/04_data_management_dementia.R now queries VNDS,
    filters to indud_kode == "U", and returns earliest haend_dato per person.
    pmin() in format_variables() applies emigration censoring automatically.

CONFIRM-5: GLP-1 exclusion in obesity comparator
  LOCATION: doc/study1_methods_plan.txt, section 3c; R/01_build_cohorts.R
  ISSUE: Persons on GLP-1 receptor agonists (ATC A10BJ) before the index date
    have already received a pharmacological intervention for obesity/T2D that
    may affect dementia risk. Including them in the obesity comparator group
    mixes treated and untreated obesity. Discussion added to methods plan
    (section 3c, Open Issues section 10). Tentative position: include in main
    analysis with GLP-1 use as a covariate; add sensitivity analysis excluding.
  ACTION: Confirm analytical decision with co-authors before implementing.

MINOR-1: Income quintiles — IMPLEMENTED (population-standardized per SEPLINE)
  STATUS: Implemented in R/03_extract_ses.R section 3.2.
  IMPLEMENTATION: 3-year average income (index_year-1, -2, -3) from FAIK,
    population quintile cutpoints computed from full BEF x FAIK population
    stratified by sex x 5-year age group x year. Statistics Denmark does not
    publish pre-computed quintile tables at this stratification level.
  CONFIRM: Run the pipeline and verify that pop_quintile_cutpoints contains
    reasonable cell counts (very small cells in rare sex x age x year strata
    may produce unstable cutpoints). If any strata have n < 100, consider
    widening to 10-year age groups for those cells.
  REFERENCE: Hjorth et al., Clin Epidemiol 2025;17:593-624, Table 9.

MINOR-3: OSDC diabetes file covers up to 2022 only
  LOCATION: R/02_extract_outcomes_covariates.R; R/04_data_management_dementia.R
  ISSUE: Persons with diabetes onset in 2023-2024 will be classified as
    No_diabetes in the OSDC file. Affects a small fraction of later surgery dates.

MINOR-4: NMI score is a modified NMI for Study 1 (excludes dementia predictors)
  LOCATION: R/02_extract_outcomes_covariates.R, extract_nmi()
  NOTE: For Study 1, exclude_dementia_predictors = TRUE removes dx_F00_G30
    and rx_N06D from the NMI computation (see CRITICAL-3 above). The resulting
    nmi_score is based on 48 predictors, not the original 50. This should be
    described explicitly in the statistical methods section. The modified score
    has not been externally validated — its predictive accuracy for mortality/
    hospitalization may differ slightly from the published NMI.
